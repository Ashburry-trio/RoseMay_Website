# Security Policy

## Supported Versions

Currently there is no official release, however there is a pre-release
work in constant development and regular Pushes to the git repo.

| Version | Supported          |
| ------- | ------------------ |
| <5.1.1   | :white_check_mark: |

## Reporting a Vulnerability

To report a vulnerability write a Issue or contact Ashburry@myproxyip.com

Goto https://github.com/Ashburry-trio/RoseMay_Website/issues and you can expect 
a security fix as soon as I check my Issues or my email inbox. I will put a push
out with the required security fix.
